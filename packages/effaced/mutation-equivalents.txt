# Equivalent-mutant allowlist for the weekly mutation hard gate (#124).
#
# Each line is a mutmut mutant name (from `uv run mutmut results`) that
# SURVIVES but provably cannot change observable behaviour — an equivalent
# mutant. The gate (scripts/check_mutation_gate.py) lets exactly these
# survive; ANY other surviving / "no tests" mutant fails the weekly job, and
# any line here that is NOT a current survivor fails it too (stale entries are
# loud — the floor must track reality). Comments (`#`) and blank lines ignored.
#
# Adding an entry: include the equivalence argument as a `#` comment above it.
# Re-baselining after sibling PRs land code in these modules: run
# `just mutmut && just mutmut-results`, then add/remove entries the gate names.
