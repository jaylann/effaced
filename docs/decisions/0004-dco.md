# 0004. DCO over CLA

- **Status:** accepted
- **Date:** 2026-06-10

## Context

Contributions must not fragment copyright (the project should stay cleanly relicensable/sellable), but a CLA bot adds friction and infrastructure for a young project.

## Decision

Developer Certificate of Origin: every commit carries `Signed-off-by` (`git commit -s`), enforced three times — local pre-commit message hook, the Claude Code commit hook, and the `dco.yml` CI check (no third-party action; a small `gh api` script).

## Consequences

- Zero contributor signup friction; standard Linux/GitLab model.
- A CLA can still be layered on later if a legal review demands it (revisit before first commercial licensing).
